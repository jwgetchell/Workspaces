1 Introduction
This document explains the features and usage of “Light Sensor ISL29177 Application”
which is developed by VVDN Technologies for Intersil Corporation.
This application will be used for isl29177 light sensor during final production, for Windows
8 platform.
1.1 Feature:
1. View continuous proximity count from the light sensor
2. Configure Light sensor for 8 different proximity sleep values
3. Configure Light sensor for 4 different proximity persistency values
4. Configure the minimum and maximum interrupt threshold of light sensor
5. Directly access to three different modes
a. Customer mode
b. Register mode
c. Engineering mode
6. Offline help for using the application
2 Installing the Application
This application is developed to be deployed on Windows xp/7/8/8.1 platform. Necessary
updates and software upgradations will be provided for the same in future releases.
Note: Make Sure you have already installed .Net framework version 3.5 or higher on your
machine.
2.1 Installation files
The archive contains following subdirectories that has the application packages for the following
computer architectures.
1. Exe file
2. Config file
2.2 Installation procedure
2.2.1 Extracting the files
 Log onto host computer running Windows operating system, using an account with local
administrator privileges.
 Copy the application package archive (VVDN_ISLU_SNSR_APP_1.0.0.1.zip) to
anywhere in the hard disk. Please use the D:\ drive for now.
